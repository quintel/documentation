#### New features

* Must-run gas turbines and combined-cycle gas turbines (CCGT) have been added to the model. These power plants run at a constant capacity. The capacity of both the must-run and dispatchable variants can be set in the <a href="https://energytransitionmodel.com/scenario/supply/electricity/gas-plants" target="_blank">Electricity</a> section.
* The 'Hydrogen supply and demand' chart has been simplified to improve readability by reducing aggregating some of the supply and demand technologies. Data per technology can still be retrieved in the <a href="https://energytransitionmodel.com/scenario/data/data_export/hourly-curves-for-hydrogen" target="_blank">Data export</a> section.

#### Bug fixes

* The maximum capacity for the <a href="https://energytransitionmodel.com/scenario/supply/hydrogen/ammonia-production" target="_blank">ammonia import</a> slider was too restrictive. This limit has now been increased.
